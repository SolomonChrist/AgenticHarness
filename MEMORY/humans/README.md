# HUMAN MEMORY

Each human gets a memory lane here.

Recommended structure for each human:

- `<HumanID>/ALWAYS.md`
- `<HumanID>/RECENT/`
- `<HumanID>/ARCHIVE/`

Example:

- `FirstnameLastname4821/ALWAYS.md`
- `FirstnameLastname4821/RECENT/2026-04-16.md`
- `FirstnameLastname4821/ARCHIVE/2026-04-summary.md`

Purpose:

- let `Chief_of_Staff` remember the operator over time
- preserve preferences, communication style, recurring priorities, and important history
- allow future `Chief_of_Staff` harnesses to inherit operator context immediately
