# AGENT MEMORY

Each role gets its own memory lane here.

Recommended structure for each role:

- `<Role>/ALWAYS.md`
- `<Role>/RECENT/`
- `<Role>/ARCHIVE/`

Example:

- `Chief_of_Staff/ALWAYS.md`
- `Chief_of_Staff/RECENT/2026-04-16.md`
- `Chief_of_Staff/ARCHIVE/2026-04-summary.md`

Purpose:

- preserve role-specific memory across harness swaps
- let new harnesses inherit the memory of the role they are taking over
- keep role-specific habits, lessons, and important standing notes searchable as markdown
